# ⚔️ 聖魂傳奇 (Holy Legend)

《**聖魂傳奇**》是一款結合 **多人即時連線** 與 **Roguelike 元素** 的網頁版爬塔 RPG 遊戲。  
玩家可以單打獨鬥，或與好友組隊挑戰無盡之塔，透過收集符文、轉職修煉與應對隨機事件，打造專屬的傳奇英雄。

---

## 🌟 核心特色

### 🏰 多人即時冒險
- **組隊系統**：支援 1–4 人即時組隊，具備隊長指派、隊員剔除與就斷線處理
- **共享商店與事件**：塔內旅行商人與神秘事件將同步全隊進度，共同決策  

### ⚔️ 深度戰鬥系統
- **回合制策略**：完整傷害計算公式，包含屬性倍率、爆擊判定與閃避機制  
- **技能石系統**：透過裝備不同符文（屬性加成／主動技能／被動增益）強化角色  
- **Buff 疊加機制**：支援多名隊友同時施放增益，依施放者 ID 判定重置時效或數值疊加  

### ⚒️ 技能工坊（Skill Forge）
- **符文裝備**：8 個圓形排列插槽，直觀管理角色戰鬥能力  
- **熔煉合成**：消耗 3 顆相同符文，有機率獲得更高階力量  
- **符文圖鑑**：完整收集後可查看詳細屬性

### 🎓 職業與成長
- **轉職體系**：多樣化職業（聖騎士、盜賊、牧師等），各自擁有獨特成長係數  
- **階級挑戰**：需滿足特定條件（例：騎士 Lv.30 + 法師 Lv.30）解鎖進階職業  
- **Roguelike 核心設計**：出塔後僅保留等級與經驗，道具與塔內屬性全部重置

## 🛠️ 技術架構

### 前端
- HTML5  
- CSS3（Tailwind CSS）  
- JavaScript（Vanilla / React）  
- Lucide Icons  

### 後端
- Node.js  
- Express  

### 即時通訊
- Socket.io（戰鬥同步、組隊與邀請系統）  

### 資料庫
- Sequelize ORM（MySQL / SQLite）  
- 多表關聯設計（User、Inventory、Equipment、Skills）  

### 檔案處理
- Multer（玩家自訂頭像上傳）

## 🚀 安裝與執行

### 1️⃣ 複製專案
```bash
git clone https://github.com/Jenne14294/HolyLegend.git
```

### 2️⃣ 安裝依賴
```bash
npm install
```

### 3️⃣ 資料庫設定
- 設定 database.js 內的連線資訊

### 4️⃣ 啟動伺服器
```bash
npm start
```

### 5️⃣ 開啟遊戲
```bash
http://localhost:3000/holylegend
```
